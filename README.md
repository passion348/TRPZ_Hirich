## Гра "Guess the number"

Популярна нині гра, де потрібно вгадати число від 0 до 100

Після кожної спроби, вам дають підказку, ваше число більше загаданого чи менше.
Всього спроб 5, але, якщо не вдалося вгадати, вам покажуть загадане число та ви зможете спробувати ще раз, але вже з новим числом

## Функціонал
- Загадування випадкового числа від 1 до 100
- Введення числа користувачем
- Підказка: більше/менше
- Лічильник спроб (максимум 5)
- Повідомлення про перемогу або програш
- Кнопка перезапуску гри з новим числом

## Технології
- HTML
- CSS
- JavaScript

## Запуск
1. Скачайте репозиторій або клонувати його
2. Відкрити файл `index.html` у браузері
3. Вводьте числа та вгадуйте!
АБО
1. Перейдіть по посиланню в цьому репозиторії та грайте прямо тут!
